<h1>📦 AGV Fleet Scheduling Optimizer — Rockwell Automation Hackathon (PICT)</h1>

<p>This project was built as part of the <strong>RokConnect Hackathon</strong> by <strong>Rockwell Automation</strong> at PICT, Pune.</p>

<h3>🛠️ Objective:</h3>
<p>
Design a dynamic and efficient scheduling system for a fleet of AGVs (Automated Guided Vehicles) to deliver payloads across 9 stations, 
while optimizing for time, load capacity, battery management, and collision avoidance.
</p>

<h3>🚗 Features:</h3>
<ul>
  <li>Robust scheduling algorithm to minimize total execution time</li>
  <li>Smart battery life and charging strategy</li>
  <li>Dynamic path planning with collision avoidance</li>
  <li>Real-time execution logs and performance metrics</li>
  <li>Prioritized payload delivery based on urgency levels</li>
</ul>

<h3>📊 Deliverables Included:</h3>
<ul>
  <li>Algorithm documentation</li>
  <li>AGV movement logs</li>
  <li>Total and average delivery times</li>
  <li>Battery usage reports</li>
</ul>
